## **1. Overview**

Oleon Chatbot is an AI-powered chat widget that you can easily integrate into your WordPress website to offer real-time support, answer user queries, and provide an enhanced user experience.

**Key Features**:

* Easy integration with WordPress.

* Fully customizable to match your website design.

* AI-driven responses for accurate information.

* Works seamlessly across devices.

***

## **2. Requirements**

Before you start, ensure the following:

* WordPress version **5.0 or above**.

* PHP version **7.4 or higher**.

* Active internet connection.

* Oleon API Key & Service ID (you will receive this after purchase or registration).

## **3. Installation**

1. Download the plugin `.zip` file from our website.

2. Log in to your WordPress dashboard.

3. Go to **Plugins > Add New > Upload Plugin**.

4. Upload the downloaded `.zip` file.

5. Click **Install Now** and then **Activate**.

***

## **6. Shortcode Integration**

You can display the Oleon Chat widget on specific pages or posts using a shortcode.

### **Shortcode Syntax**

```
[oleon_chat position="right" theme="dark" base_color="#007bff"]
```

### **Parameters**

| Parameter    | Values          | Default   | Description                         |
| ------------ | --------------- | --------- | ----------------------------------- |
| `position`   | `left`, `right` | `right`   | Set widget position.                |
| `theme`      | `light`, `dark` | `dark`    | Choose the widget theme.            |
| `base_color` | Hex color codes | `#007bff` | Customize the primary widget color. |

### **Example**

To show the chat widget in light mode on the left:

```
[oleon_chat position="left" theme="light" base_color="#ff5722"]
```

## **7. Troubleshooting**

### **Common Issues & Solutions**

1. **Widget Not Appearing**

   * Ensure the plugin is **activated** in the Plugins section.

   * Verify your API Key is correctly added under **Settings > Oleon Chatbot**.

2. **Incorrect API Key**

   * Double-check your API Key on the [Olee AI Dashboard](https://olee.ai).

   * Copy and paste it without any extra spaces.

3. **CSS Conflicts**

   * Use the **Custom CSS** field to override styles.

If the issue persists, check the [Support Section](#support).
